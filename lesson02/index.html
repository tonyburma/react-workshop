React Essentials

What is React?

  • Component-based Liberary for building user interfaces
  • Manages the state of applications efficiently
  • Developed and maintained by Meta Inc (formerly Facebook)
  • Open-source and widely adopted in the web development community

Notes

React is a JavaScript library for building user interfaces. It allows developers to create reusable UI components and manage the state of their applications efficiently.

Why Use React?

  • Fast: Virtual DOM
  • Large community support (Open source)
  • Easy to create Single Page Applications (SPAs)

Notes

React's declarative syntax makes it easy to understand and predict how the UI will look at any given time. The component-based architecture allows for better code organization and reusability.

Notes

JSX is a syntax extension for JavaScript that looks similar to HTML. It allows developers to write HTML-like code within their JavaScript files, making it easier to create and manage UI components.

React Components

  • Basic building blocks of React applications
  • Every piece of UI is a component
  • Components can be nested inside other components
  • Define components using functions or classes (Functional and Class components)
  • Since React 16.8, functional components are preferred (we will use functional components only)

Prepare Development Environment

Pre-requisite

You need to have Node.js and npm (Node Package Manager) installed on your machine.

You can download Node.js from the official website: Node.js

After installing Node.js, you can check if npm is installed by running the following command in your terminal: 

              npm -v

If you see a version number, it means npm is installed successfully.

Setting Up Your Development Environment

To start building React applications, you need to set up your development environment. You can use tools like Create React App, or Vite (Vite, pronounced /vÉªt/ (like "veet")) to quickly set up a new React project.
