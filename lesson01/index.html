Type Coercion

JavaScript automatically converts types when needed. This is called type coercion. (kōˈərSH(ə)n) 

            
              1 + true; // 2
              1 + "1"; // "11"
              console.log("2" == 2); // true
              1 / "10"; // 0.1
              "10" * "2"; // 20
              10 * 'a'; // NaN

Comparison

JavaScript provides several operators for comparison.

            
              console.log("2" == 2); // true
              console.log("2" === 2); // false (strict comparison)
              console.log("2" != 2); // false
              console.log("2" !== 2); // true (strict comparison)
              console.log("2" > 2); // false
              console.log("2" < 2); // false
              console.log("2" >= 2); // true
              console.log("2" <= 2); // true

Functions

Declaration

Functions are first-class objects in JavaScript. They can be assigned to variables, passed as arguments, and returned from other functions. 

            
              // Named function
              function add(a, b) {
                return a + b;
              }
              let sum = add(2, 3);
              console.log(sum); // 5

              // Anonymous function
              const multiply = function(a, b) {
                return a * b;
              };
              let product = multiply(2, 3);
              console.log(product); // 6

Functions

Arrow Function
            
              // Anonymous function
              const add = function(a, b) {
                return a + b;
              }

              // Arrow function 
              const add = (a, b) => {
                return a + b;
              };

              // Shorter syntac 
              const add = (a, b) => a + b;

              let sum = add(2, 3);
              console.log(sum); // 5

Objects

  • JS is a multi-paradigm language. It supports object-oriented, functional, and imperative programming styles.
  • Possible to create objects using the "new" keyword or using object literals.
  • Object definition (class) is defined using functions
  • "this" keyword can be used to assign instance variables.

Objects

Object Definition (class)
            
              // Build-in object
              console.log(new Date()); // current date and time 

              // Class definition
              function Person(name, age) {
                this.name = name;
                this.age = age;
              }

              const alice = new Person("Alice", 25);
              console.log(typeof alice); // object
              console.log(alice instanceof Person); // true

              // Try it Out
              const alice1 = new Person("Alice", 25);
              console.log(typeof alice1);

Arrow functions do not have their own "this" context. So we cannot use arrow functions to define a class.

Objects

Object Literals
            
              const alice = {
                name: "Alice",
                age: 25,
                greet: function() {
                  console.log("Hello, my name is " + this.name);
                }
              };

              console.log(typeof alice); // object
              alice.greet(); // Hello, my name is Alice

Arrays

  • Create array using "new Array()"" or Array Literal.
  • An array can consist of different types. 
            
              const arr1 = new Array(1, 2, 3);
              const arr2 = [1, "two", true, null];

              console.log(arr1); // [1, 2, 3]
              console.log(arr2); // [1, "two", true, null]
              console.log(typeof arr1); // object
              console.log(typeof arr2); // object
              console.log(Array.isArray(arr1)); // true
              console.log(Array.isArray(arr2)); // true

Add Items to Arrays

  • push( ) - add to the end
  • unshift( ) - add to the beginning
  • splice( ) - add at a specific index
  • concat() - merge two arrays
  • spread operator - merge two arrays
            
              const arr = [1, 2, 3];
              arr.push(4);
              console.log(arr); // [1, 2, 3, 4]
            
          
            
              const arr = [1, 2, 3];

Remove Items from Arrays

  • pop() - remove from the end
  • shift() - remove from the beginning
  • splice() - remove at a specific index
  • slice() - create a new array with a portion of the original array
  • filter() - create a new array with elements that pass a test

Commonly Used Array Methods

  • forEach( ) - iterate over each element
  • map( ) - create a new array with the results of calling a function on each element
  • filter( ) - create a new array with elements that pass a test
  • find( ) - return the first element that satisfies the provided testing function

forEach( )

The forEach() method executes a provided function once for each array element. 

            
              const numbers = [1, 2, 3, 4, 5];
              numbers.forEach(function(number) {
                console.log(number);
              });

map( )

The map() method creates a new array with the results of calling a function for every array element. 

            
              const numbers = [1, 2, 3, 4, 5];
              const doubled = numbers.map((number) => {
                return number * 2;
              });
              console.log(doubled);

fileter( )

The filter() method creates a new array with all elements that pass the test implemented by the provided function. 

            
              const numbers = [1, 2, 3, 4, 5];
              const evenNumbers = numbers.filter((number) => {
                return number % 2 === 0;
              });
              console.log(evenNumbers);

find( )

The find() method returns the first element that satisfies the provided testing function. 

            
              const numbers = [1, 2, 3, 4, 5];
              const found = numbers.find((number) => {
                return number > 3;
              });
              console.log(found);

Arrow Functions

Shorter syntax for writing functions. The syntax is a little similiar to Lambda expressions in other languages such as C# and Python. 

            
              // Regular function
              const add = function (a, b) {
                return a + b;
              }

              // Arrow function
              const square = (x) => x * x;
              console.log(square(4)); // 16

              // () is optional if there is only one parameter
              const numbers = [1, 2, 3, 4, 5];
              
              forEach(number => {
                console.log(number);
              });

Template Literals

Template literals are string literals that allow embedded expressions. You can use multi-line strings and string interpolation features with them. 

            
              const name = "Alice";
              const greeting = `Hello, ${name}!`;
              console.log(greeting); // Hello, Alice!
            
          
            
              function greet(gender, name) {
                return `Hello ${gender === "M" ? "Mr. " : "Ms. "} ${name}`;
              }

              console.log(greet("M", "Smith")); // Hello Mr. Smith
              console.log(greet("F", "Molly")); // Hello Ms. Molly

Destructuring Assignment

Destructuring assignment is a syntax that allows unpacking values from arrays or properties from objects into distinct variables. 

            
              const person = {
                name: "Alice",
                age: 25,
                city: "New York"
              };

              const { name, age } = person;
              console.log(name); // Alice
              console.log(age); // 25

              // Renaming variables
              const { name: firstName, age: years } = person;
              console.log(firstName); // Alice
              console.log(years); // 25

Spread and Rest Operators

Spread Operator

The spread operator (...) allows an iterable such as an array to be expanded in places where zero or more arguments are expected. (Expanding) 

            
              const numbers = [1, 2, 3];
              const newNumbers = [...numbers, 4, 5];
              console.log(newNumbers); // [1, 2, 3, 4, 5]

              const person = {
                name: "Alice",
                age: 25
              };

              const newPerson = { ...person, city: "New York" };
              console.log(newPerson); // { name: "Alice", age: 25, city: "New York" }

Spread and Rest Operators

Rest Operator

The rest operator (...) allows us to represent an indefinite number of arguments as an array. (Collecting) 

            
              function sum(...numbers) {
                return numbers.reduce((acc, num) => acc + num, 0);
              }

              console.log(sum(1, 2, 3)); // 6
              console.log(sum(1, 2, 3, 4, 5)); // 15
              console.log(sum(1, 2, 3, 4, 5, 6)); // 21

Default Parameters

Default parameters allow named parameters to be initialized with default values if no value or undefined is passed. 

            
              function multiply(a, b = 1) {
                return a * b;
              }

              console.log(multiply(5)); // 5
              console.log(multiply(5, 2)); // 10

Promises and async/await

Promises are used to handle asynchronous operations in JavaScript. The async/await syntax makes it easier to work with promises. 

            
              fetch("https://jsonplaceholder.typicode.com/photos")
                .then(response => {
                  if (!response.ok) {
                    throw new Error("Network response was not OK");
                  }
                  return response.json();
                })
                .then(data => {
                  console.log("Data loaded with Promise:");
                  console.log(data.slice(0, 5)); // Show first 5 items
                })
                .catch(error => {
                  console.error("There was a problem with the fetch operation:", error);
                });

Promises and async/await

The async/await syntax makes it easier to work with promises. 

            
              async function fetchData() {
                try {
                  const response = await fetch("https://jsonplaceholder.typicode.com/photos");
                  if (!response.ok) {
                    throw new Error("Network response was not OK");
                  }
                  const data = await response.json();
                  console.log("Data loaded with async/await:");
                  console.log(data.slice(0, 5)); // Show first 5 items
                } catch (error) {
                  console.error("There was a problem with the fetch operation:", error);
                }
              }

              fetchData();
