Node.js and npm

Node.js is a JavaScript runtime built on Chrome's V8 JavaScript engine. It allows you to run JavaScript on the server side.

npm (Node Package Manager) is a package manager for JavaScript. It allows you to install and manage packages (libraries) for your projects. It is included with Node.js installation.

You can install Node.js and npm from the official website: nodejs.org

Check Installation

After installation, you can check the versions of Node.js and npm using the following commands in your terminal: 

            
              node -v
              npm -v

You can also use the following command to check if npm is installed correctly: 

            
              npm --version

HTML and CSS

Structure of HTML Document

  • <!DOCTYPE html>
  • <html lang="en">
  • <head>
  • <body>

The <!DOCTYPE html> declaration defines the document type and version of HTML being used. The <html> element is the root element of an HTML page, and the lang attribute specifies the language of the document.

The <head> element contains meta-information about the document, such as the title and links to stylesheets. The <body> element contains the content of the document that is displayed in the browser.

HTML Document

            
              <!DOCTYPE html>
              <html lang="en">
                <head>
                  <meta charset="UTF-8">
                  <meta name="viewport" content="width=device-width, initial-scale=1.0">
                  <title>Document</title>
                </head>
                <body>
                  <h1>Hello World</h1>
                </body>
              </html>

The <meta> tag with the charset attribute specifies the character encoding for the HTML document. The <title> tag defines the title of the document, which is displayed in the browser tab.

The <h1> tag defines the main heading of the document. The content inside the <body> tag is what is displayed in the browser.

CSS Basics

  • Selectors
  • Properties
  • Values

CSS (Cascading Style Sheets) is used to style HTML documents. It allows you to apply styles to elements based on their selectors, such as class, ID, or element type.

CSS properties define the styles to be applied, such as color, font-size, margin, padding, etc. Values specify the specific style to be applied to the property.

CSS Example

            
              body {
                background-color: "red";
                font-family: Arial, sans-serif;
              }
              h1 {
                color: #333;
              }

In this example, the body element has a "red" background color and uses the Arial font. The h1 element has a dark gray text color.

CSS can be included in HTML documents using the <style> tag in the <head> section or by linking to an external CSS file using the <link> tag. 

            
              <link rel="stylesheet" href="styles.css">
